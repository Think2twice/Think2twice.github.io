课程首页 12. Workspace seeds 要能重复安装和回滚

Workspace Seeds

配置也要进版本管理

这一课对应 workspace seed 提交：把 prompts、skills、tools 和 action functions 变成可校验、可安装、可回滚的文件包。

核心判断

工作空间资源是产品能力的一部分。只存在数据库里的配置，不能复盘、不能 diff，也很难迁移。

安装器边界

API 优先

通过 OpenWebUI API 安装，不直接改数据库。

可校验

`--validate-only` 只读检查 seed 文件格式。

可回滚

`--uninstall` 只删除 seed 包管理的资源。

不污染

Action functions 默认不全局启用。
