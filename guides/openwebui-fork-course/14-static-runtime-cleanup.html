课程首页 14. 清理旧静态描述文件

Static Descriptors

删文件之前，先确认谁还在引用它

这一课对应静态描述文件清理提交：`manifest.json` 和 `opensearch.xml` 已经由后端动态路由接管，旧 OpenWebUI 静态描述文件就应该从 fork 里退场。

核心判断

旧描述文件不是因为“看起来没用”才删除，而是因为前端入口仍保留，实际响应已经交给后端动态路由生成品牌化内容。

手工检查顺序

先查路由

确认 main.py 已经提供 manifest 和 opensearch 动态接口。

再看旧内容

静态 opensearch 里仍写着旧品牌时，就应该退出权威入口。

只删描述

只删除 tracked 描述文件，不误删图片占位符、marker 和 robots。

最后验证

跑 diff 检查、HTML smoke 和 staged 文件私密路径扫描。
