字幕转换
触发：用户说“RT字幕”且给出 ASS 文件。动作：按上下文纠正为 SRT，转换并抽查条数。验证：原文件不改，输出 SRT 可读。
下载权限
触发：直接打开存储 API 链接返回权限不足。动作：判断是私有存储授权，不是文件不存在。验证：网页 Storage 下载或带临时凭据下载。
Mac 负载
触发：用一段时间后开始卡。动作：把 WindowServer、语音助手、Typeless、浏览器/Electron、录屏悬浮工具分层。验证：看进程联动而不是盯单一 App。
Pages 404
触发：本地提交后线上页面 404。动作：比对本地/远端提交、线上 posts.js、页面响应。验证：区分构建延迟、缓存和路径错误。
