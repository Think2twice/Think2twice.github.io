工具与服务器散沙合集

AList 缓存、服务器清理和视频 Skill 分类，本质都是先分层再动手。

收进来的事项

AList 目录缓存

备份数据库后只调整目录结构缓存，不碰账号、凭据、挂载路径和视频代理。

旧部署清理

确认新服务和真实数据卷后，再清理旧容器、旧镜像和无用卷。

视频 Skill 分类

按输入对象和生产阶段分类，而不是按“哪个更强”比较。

验证方式

性能优化看前后耗时，清理看磁盘空间和服务健康，分类看是否能指导下次选择。

可复用规则

  • 服务器操作先保护当前线上服务和数据卷。
  • 缓存优化先备份，再只动最小配置。
  • 视频工具选择看任务阶段：生成、批量模板、口播精剪、素材编辑、资产生成。
