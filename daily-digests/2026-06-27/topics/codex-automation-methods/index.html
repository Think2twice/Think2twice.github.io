每日复盘发布

Agent 邮箱

邮箱能力只覆盖邮件收发；目标站是否能自动注册，还要看实名、验证码、风控和服务条款。

GitOps-lite

本地源码是权威，服务器是运行面，构建产物从源码生成，数据卷不是源码。

公开页元数据

每日复盘写入 posts.js，但保持 Archive 可见、首页隐藏。

可复用规则

  • 先判断任务所在层：聊天记录、源码、构建产物、服务器、数据卷。
  • 自动化任务必须写清触发、动作、验证和停止条件。
  • 公开沉淀只写流程，不写真实凭据、账号、私密入口和原聊天。
