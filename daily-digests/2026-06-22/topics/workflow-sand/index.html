散沙合集

这些不是大项目，但都值得进总账，后续可能沉淀为小检查单。

工具

K380 与 Typenus

触发：外接键盘快捷键担心影响内置键盘。动作：区分 K380 Fn、macOS Globe/Fn、Typenus 全局快捷键和按设备 remap。验证：优先只映射外接键到 F18。

系统

Mac 磁盘与卡顿

触发：系统明显卡顿且可用空间低。动作：先清缓存、Xcode 可重建内容、安装包和应用缓存。验证：空间恢复后停在真实数据边界，不动 Docker/照片/聊天文件。

后台

Adobe 启动项

触发：用户明确不用 Creative Cloud。动作：禁用用户级启动项和当前进程。验证：复查无用户级 Adobe Creative Cloud 进程，系统级 daemon 保留待管理员密码或正式卸载。

敏感

API 网关配置

触发：图片生成链路和配置需要排查。动作：只记录“后端兜底识别图片意图”和“敏感配置核对”。验证：公开页不出现 key、账号或真实入口。

Pages

电影 Ticket Box

触发：本地 HTML 要进入 GitHub Pages。动作：发布到 guides，补海报/片名外链。验证：线上锚点数量和 Pages build 状态。

自动化

每日页重写

触发：同日目录已存在但覆盖窗口错位。动作：按真实窗口重写。验证：总账、专题、posts.js excerpt 和覆盖说明一致。

可复用结论

  • 系统维护类任务先分风险层：可重建缓存、用户确认项、真实数据禁动项。
  • 外设快捷键默认按设备隔离，而不是改全局快捷键。
  • 公开复盘里，任何 API/服务器/账号相关内容先降级成过程类型，再决定是否能写细节。
