媒体与设备工具散沙合集

视频伪装头修复、录屏变速、Windows C 盘清理和启动项残留，都是“先识别结构，再给保守修复”的工具流。

返回本日总账

今天收进来的事项

  • 伪装 MP4：检查文件头，去掉伪装字节，保留原件。
  • 录屏变速：分段处理，验证新视频时长和可读性。
  • Windows C 盘：先读 SpaceSniffer 快照，只建议清缓存、更新残留和系统工具可处理项。
  • 启动项残留：查 Run/RunOnce 和启动文件夹，不碰个人资料。

可复用规则

工具类问题不要先猜格式或一键删除。先识别结构，再选最保守动作；只要涉及别人电脑，默认不删个人文件和不可逆数据。
