媒体文件、Windows 清理和启动项残留：小问题也要有安全边界

这些问题都能用命令快速解决，但 Fay 的复盘重点不是命令本身，而是“先判断结构，再给可逆方案，再留日志”。

伪装视频修复

视频不能播放时，不急着改后缀。先看文件头、容器和流信息。当天样本是前面塞了伪装字节，移除后可播放；画质差来自源文件参数，不是封装问题。

中段倍速

视频编辑任务要先明确时间段、倍速、其余片段处理方式和输出格式。完成后用时长、编码信息和可读性确认，而不是只看文件存在。

C 盘清理

别人电脑清理不能暴力删除。先读 SpaceSniffer 快照，分安全清理、谨慎清理、不建议动；脚本要写日志、避开系统关键目录，并保留人工确认点。

启动项残留

开机找不到某个卸载程序，常见原因是注册表、启动目录或计划任务残留。处理目标是删除启动引用，不是继续找已经不存在的文件。

可复用检查清单

  • 媒体文件：先查文件头和容器，再决定修复、转封装或重编码。
  • 别人电脑：先快照，再脚本日志，再人工确认。
  • Windows 清理：系统目录只用系统工具，不手删核心目录。
  • 启动项问题：注册表、启动目录、计划任务三处一起查。
