AI Notes 发布准备

AI Notes 已经不只是功能原型。0.1.0 release 链路覆盖构建、真实扩展来源、三站 fixture、长文、附件、导出、备份恢复、工作台布局、选区工具条、大纲结构、权限审计和 zip 打包。

已具备

  • Chrome MV3 扩展骨架和三站按钮注入。
  • TipTap/ProseMirror 活文档、导出、附件、公式、表格、图片和大纲。
  • 本地备份/恢复，恢复冲突默认生成副本。
  • 发布包、图标、商店截图草稿和权限说明。

仍缺

  • Fay 可视 Chrome 手动加载 dist 并核对构建胶囊。
  • ChatGPT、Gemini、Claude 真页面按钮、摘录和验收中心报告。
  • 正式隐私政策公开 URL、支持邮箱、商店文案审稿。
  • WebDAV 冲突 UI 和兼容性矩阵。
